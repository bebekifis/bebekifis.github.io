Image Retrieval via Deep Neural Network

Yinan Yu@IDL

This page shows some results of visual similarity image retrieval via Deep Neural Network. It is not a toy, but our online commercial service (http://stu.baidu.com, click "推荐相似图片"). The back-end database contains about 1 billion images. The response time of each query is about 0.7 sec (600 images).

Content-based Image Retrieval (CBIR) is an old and so old topic. Given a query image, we extract its representation via Deep Neural Network (DNN) and use it to retrieve the most similar images. The architecture is robust to illumination, color, style, coarse structure, main contains, texture, etc.

Fortunately, it works, but no one knows why. Please don't expect too much about the result
